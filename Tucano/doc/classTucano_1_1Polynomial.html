Detailed Description

A polynomial function of arbitrary degree, usually fit from given data.

The polynomial can also be set by hand, by setting its parameters, but usually it is defined by a simple regression method to fit given data

Note that our polynomial is defined in 3D space. First fit a plane to the points, then find the polynomial curve on the plane.
