Detailed Description

Class for object manipulator.

Extends trackball class to work as an object manipulator, it should always envelop a specific mesh and be used to manipulate it (translation and rotation).

This is an abstract class, should instance one of the inherited classes

Typically, it receives a pointer to the main camera, so it can also act in camera space according to the mesh and the camera

The manipulator is visually represented as a sphere
