Eigen Primer

In Tucano all linear algebra operations and transformations are handled by the Eigen library. Here are a few examples to help you get started and complete the exercises. But keep in mind that Eigen is a very powerful library, and the geometry module is just a fraction of it.

Note: we strongly advise you to checkout the Space Transformation Tutorial from the Eigen webpage. But here is a very primer on how you can work with matrices.


Matrices and Vectors

Create a 4x4 matrix in Eigen and set it as the identity matrix:

Eigen::Matrix4f mymatrix = Eigen::Matrix4f::Identity();

Create a 4x4 matrix in Eigen and set all values to zero:

Eigen::Matrix4f mymatrix = Eigen::Matrix4f::Zero();

Create 3x1 and 4x1 Vectors and set them to zero:

Eigen::Vector3f my3vec = Eigen::Vector3f::Zero();
Eigen::Vector4f my4vec = Eigen::Vector4f::Zero();

You can do any sort of operation with your matrices and vectors, for example:

Eigen::Vector3f mynew4vec = mymatrix * my4vec;

Note: for more examples, see the Eigen Matrix and Vector Arithmetic Tutorial

std::cout << "This is my matrix" << std::endl << mymatrix << std::endl;


Affine transformations

Affine transformations in Eigen are represented by the AffineMatrix class. You learned in the lecture on homogeneous coordinates that 3D transformations may be represented by 4x4 matrices. The Affine 3D matrix in fact holds internally a 4x4 matrix, but it allows to easily perform affine transformation such as scaling, translation, and rotation without having to write all matrices by hand. So you can think of it as an extension of the Matrix4f class. Look at some usage examples below:

Create an Affine 3D matrix:

Eigen::Affine3f myaffinematrix = Eigen::Affine3f::Identity();

If you actually need the 4x4 matrix stored inside the Affine3f, you can retrieve like this:

Eigen::Matrix4f my4x4matrix = myaffinematrix.matrix();

Apply a rotation to the affine matrix using Angle Axis notation, that is, you pass an axis of rotation and the angle. In this case we will rotate 45 degrees around the Y axis.

myaffinematrix.rotate (Eigen::AngleAxisf (M_PI*0.25, Eigen::Vector3f::UnitY()));

Apply a translation, in this case 0.5 unit along the X axis. Note that you do not need a 4x1 vector, since it will assume the last element w is equal to 1.

myaffinematrix.translate (Eigen::Vector3f(0.5, 0.0, 0.0));


Printing the matrices

If you are curious to see the resulting matrix or vector, you can print them using the cout interface! The following lines will create, rotate and print the resulting matrix:

Eigen::Affine3f myaffinematrix = Eigen::Affine3f::Identity();
myaffinematrix.rotate (Eigen::AngleAxisf (M_PI*0.25, Eigen::Vector3f::UnitY()));
std::cout << myaffinematrix.matrix() << std::endl;

The output is:

0.707107 0 0.707107 0
0 1 0 0
-0.707107 0 0.707107 0
0 0 0 1
